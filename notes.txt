=== INSTALLATION ===

1) Edit the connect.inc.php in the root folder to have the correct username, password and database name for your database (Can usually be found on the CPanel if you have one).

2) Go to your SQL terminal, paste the SQL code below and press "Go"

3) Copy and paste ALL files and folders located in the root folder (earp) into the location you want the CAD (probably cad.ea-rp.com)
